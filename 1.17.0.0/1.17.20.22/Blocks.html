方块组件

名称 类型 默认值 描述
minecraft:block_light_absorption 整数 0 定义该方块吸收的亮度变化量
minecraft:block_light_emission 小数 0.0 定义该方块发光的亮度，定义范围为[0.0, 1.0]
minecraft:breakonpush 布尔值 定义该方块被推动时是否被活塞破坏
minecraft:breathability 枚举 solid 定义方块的可呼吸性以及其是会被当做固体还是气体
minecraft:destroy_time 小数 0.0 设置方块的摧毁时间，数值越大挖掘时间越长
minecraft:display_name 字符串 指定方块的显示名称ID
minecraft:entity_collision 布尔值 false 当设定为false时会禁用该方块与实体的碰撞箱
名称 类型 默认值 描述
origin 数组 [-8.0, 0.0, -8.0] 碰撞箱边界的最小起始位置
size 数组 [16.0, 16.0, 16.0] 该组件碰撞箱每一个方向上的大小
minecraft:explosion_resistance 小数 0.0 定义方块的爆炸抗性
minecraft:flammable JSON对象 定义方块的燃烧属性
名称 类型 默认值 描述
burn_odds 整数 0 定义该方块着火时被烧掉的概率。值必须大于等于0。
flame_odds 整数 0 定义该方块靠近火源时着火的概率。值必须大于等于0。
minecraft:friction 小数 0.1 定义方块的摩擦力，摩擦力会影响实体在该方块上行走的效果。
minecraft:geometry 字符串 填写要使用的几何定义名
minecraft:immovable 布尔值 定义该方块能否不能被活塞推动
minecraft:map_color 字符串 用十六进制颜色定义该方块在地图上显示的颜色
minecraft:material_instances JSON对象 将几何文件中的一个面或material_instance名映射到真实的材质实例上。填入的材质实例可以是一个完整的材质实例，也可以是另一个已经定义好的实例名
minecraft:onlypistonpush 布尔值 拥有该组件的方块不能被黏性活塞拉回
minecraft:pick_collision 布尔值 false 当设定为false时会禁用该方块与实体的碰撞箱
名称 类型 默认值 描述
origin 数组 [-8.0, 0.0, -8.0] 碰撞箱边界的最小起始位置
size 数组 [16.0, 16.0, 16.0] 该组件碰撞箱每一个方向上的大小
minecraft:placement_filter JSON对象 设置该方块在什么条件下能够被放置和生存的规则

条件

该方块能够被放置和生存的条件的列表

名称 类型 默认值 描述
allowed_faces 数组 包含下列任意字符串的列表: up，down，north，south，east，west，side，all
block_filter 数组 该方块可以在allowed_faces的方向上置于其上的方块（可使用标签指定）的列表

minecraft:preventsjumping 布尔值 该组件会使行动对象在上方行走时无法跳跃
minecraft:random_ticking JSON对象 该组件将在两值为端点的固定区间内触发事件
名称 类型 默认值 描述
Trigger on_tick
名称 类型 默认值 描述
condition 字符串 1 在该方块上执行事件的条件
event 字符串 set_block_property 将要执行的事件类型
range 数组 [10, 10] 组件将会触发事件的范围区间
target 字符串 self 执行事件的目标对象
minecraft:rotation 矢量 [a, b, c] 旋转角[0, 0, 0] 方块围绕中心旋转的角度，顺序为x-y-z
minecraft:ticking JSON对象 该组件将在两个定义值之间的固定区间内触发事件
名称 类型 默认值 描述
Trigger on_tick
名称 类型 默认值 描述
condition 字符串 1 对该方块执行事件的条件
event 字符串 set_block_property 将要执行的事件
range 数组 [10, 10] 组件将会触发事件的范围区间
target 字符串 self 执行事件的对象
looping 布尔值 true 该组件是否生成掉落物
range 数组 [10, 10] 组件将会触发事件的范围区间
minecraft:unit_cube 字符串 定义用于连接的模型块
minecraft:unwalkable 布尔值 false Sets the block as unwalkable. Mobs would not attempt to path over top of it when the value is set to true.

方块定义属性

这些属性是方块定义的一部分，决定了系统如何解析和初始化该方块。

名称 类型 默认值 描述
format_version 字符串 说明该方块被定义时的游戏版本，如果低于当前版本，将应用在定义版本中的所有修改于该方块

方块事件响应

方块触发器组件的事件响应

名称 类型 默认值 描述
add_mob_effect JSON对象 对目标添加生物效果
名称 类型 默认值 描述
amplifier 整数 0 生物效果的等级
duration 小数 0.0 生物效果的时长
effect 字符串 添加的生物效果
target Minecraft滤器 self 被执行的目标
damage JSON对象 对目标造成伤害
名称 类型 默认值 描述
amount 整数 0 造成的伤害量
target Minecraft滤器 self 被执行的目标
type 字符串 造成的伤害类型
decrement_stack JSON对象 将手持物品减一
die JSON对象 杀死目标。如果目标是self并且由方块执行则破坏该方块
名称 类型 默认值 描述
target Minecraft滤器 self 被执行的目标
play_effect JSON对象 在目标位置生成粒子效果
名称 类型 默认值 描述
data 整数 0 粒子效果数据值
effect 字符串 粒子效果名称
target Minecraft滤器 self 被执行的目标
play_sound JSON对象 在目标位置播放声音
名称 类型 默认值 描述
sound 字符串 播放声音字段
target Minecraft滤器 self 被执行的目标
remove_mob_effect JSON对象 从目标身上移除生物效果
名称 类型 默认值 描述
effect 字符串 移除的生物效果，使用“all”来移除所有效果
target Minecraft滤器 self 被执行的目标
run_command JSON对象 触发指令或一个指令组
名称 类型 默认值 描述
command 字符串 触发的指令
command array 数组 触发的指令组
target Minecraft滤器 self 被执行的目标
set_block JSON对象 将该方块变为其他方块
名称 类型 默认值 描述
block_type 字符串 更改的方块名称
set_block_at_pos JSON对象 在该方块的关联坐标放置另一方块
名称 类型 默认值 描述
block_offset 向量 [a, b, c] [0.0, 0.0, 0.0] 距离方块中心的偏移量
block_type 字符串 更改的方块名称
set_block_property JSON对象 设置该方块的property
名称 类型 默认值 描述
property MoLang 设置的property
spawn_loot JSON对象 从方块上生成掉落物
名称 类型 默认值 描述
table 字符串 文件路径，和行为包路径关联，到战利品表路径
swing JSON对象 使目标执行“swing”动画
teleport JSON对象 将目标随机传送到范围内地点
名称 类型 默认值 描述
avoid_water 布尔值 true 确定是否避免将目标传送至水中。
destination 向量 [a, b, c] [0.0, 0.0, 0.0] 传送的起始地
land_on_block 布尔值 true 确定是否将目标传送至方块上
max_range 向量 [a, b, c] [8.0, 8.0, 8.0] 目标离传送起始地能够传送的最大范围
target Minecraft滤器 self 被执行的目标
transform_item JSON对象 将物品转换为另一物品
名称 类型 默认值 描述
transform 字符串 转变的物品名称
